# EH-Downloader : download via .zip

This is API to help download gallery in EH

## How to use
1. Coppy url EH as you wants to
3. Add your EH url as https://en-downloader.herokuapp.com/?download=${YOUR_EH_GALLERY_URL}
4. Try to check {"status"} must be "ok" and {"inprogress"} must be "false"
5. If as be at number (4.) you can download by https://en-downloader.herokuapp.com/zip?download=${YOUR_EH_GALLERY_URL}
